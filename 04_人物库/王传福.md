# 王传福

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | Wang Chuanfu |
| 中文名 | 王传福 |

| 职位 | 比亚迪创始人兼董事长/总裁 |
| 学历 | 中南大学工学硕士 |

## 创业经历

### 创立比亚迪
- 1995年创立比亚迪
- 最初做手机电池
- 2003年进入汽车行业
- 2005年推出第一款电动汽车F3

## 主要成就

### 1. 电池技术突破
- 带领比亚迪成为全球最大动力电池生产商
- 突破磷酸铁锂电池技术
- 成本领先全球

### 2. 垂直整合模式
- 建立从电池到整车的全产业链
- 降低成本，提高效率
- 快速响应市场

### 3. 新能源领导者
- 比亚迪新能源汽车销量全球第一
- 业务覆盖全球50多个国家和地区

## 获得荣誉

- 2009年胡润百富榜中国首富
- "改革开放40年百名杰出民营企业家"
- 中国工程院院士（2017年）

## 巴菲特评价
