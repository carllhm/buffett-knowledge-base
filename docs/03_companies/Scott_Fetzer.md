# Scott & Fetzer

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | The Scott & Fetzer Company |
| 行业 | 多元化制造 |
| 总部 | 俄亥俄州 |
| 伯克希尔持股 | 100%（1986年收购） |
| 收购价格 | 约3.2亿美元 |

## 核心业务

旗下拥有多个业务单元：
- **世界图书**（World Book）：美国第一百科全书
- **Kirby**：高端吸尘器直销
- **Campbell Hausfeld**：空气压缩机
- 其他工业业务

## 收购背景

### 1985年底收购
- 伯克希尔以约3.2亿美元收购整家公司
- CEO Ralph Schey掌权九年，记录出色
- 价格合理，业务质量高

### 收购逻辑
1. **业务组合优秀**：世界图书和Kirby都是行业领先
2. **管理能力**：Ralph Schey运营资本配置能力superb
3. **定价权**：产品有差异化，不靠价格战
4. **现金牛**：产生稳定现金流

## 核心竞争优势

### 1. 世界图书（World Book）
| 特点 | 内容 |
|------|------|
| 市场地位 | 美国第一百科全书 |
| 销售模式 | 直销（上门销售） |
| 产品特点 | 编辑方法聪明，难度递进 |
| 1986年表现 | 销量比1982年涨45% |

### 2. Kirby吸尘器
| 特点 | 内容 |
|------|------|
| 市场地位 | 高端吸尘器领先品牌 |
| 产品特点 | 耐用，三四十年的还在用 |
| 价格定位 | 贵但最好用 |
| 1986年表现 | 全球销量比1982年涨33% |
