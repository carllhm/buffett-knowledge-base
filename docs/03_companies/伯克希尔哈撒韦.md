# 伯克希尔·哈撒韦（Berkshire Hathaway）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | Berkshire Hathaway Inc. |
| 行业 | 多元化投资控股 |
| 成立 | 1839年（纺织厂），1965年（巴菲特控制） |
| 总部 | 内布拉斯加州奥马哈 |
| 股票代码 | NYSE: BRK.A, BRK.B |

## 发展历程

### 纺织时代（1839-1985）
- 1839年：河谷瀑布公司（纺织厂）
- 1955年：伯克希尔与哈撒韦合并
- 1965年：巴菲特取得控制权（股价约$12）
- 1985年：关闭纺织业务

### 转型时代（1967-1980年代）
| 年份 | 事件 |
|------|------|
| 1967 | 收购国家赔偿公司（保险） |
| 1972 | 收购喜诗糖果（通过蓝筹印花） |
| 1983 | 蓝筹印花与伯克希尔合并 |
| 1985 | 关闭纺织，全面转向投资 |
| 1985 | 投资首都城市/ABC（$517.5M） |
| 1987 | 投资所罗门兄弟（$700M优先股） |

### 财务里程碑
| 年份 | 每股净值 | 年复利增长 |
|------|---------|-----------|
| 1965 | $4 | - |
| 1985 | $1,200+ | 约22% |
| 1987 | $2,477 | 23%+ |

## 核心业务板块

### 1. 保险业务
- **国家赔偿公司**：1967年收购，承保纪律严格
- **GEICO**：约35-41%持股，最大非控股持仓
- **其他保险子公司**：Fireman's Fund份额参与等

### 2. 实业控股
| 公司 | 行业 | 收购年份 |
|------|------|---------|
| 喜诗糖果 | 糖果零售 | 1972 |
| 内布拉斯加家具城 | 家具零售 | 1983 |
| 水牛城新闻 | 报纸 | 1977 |
| Scott & Fetzer | 多元制造 | 1985 |
| Fechheimer Bros. | 制服 | 1986 |

### 3. 股权投资
- **华盛顿邮报**：1973年买入
- **首都城市/ABC**：1985年买入
- **所罗门兄弟**：1987年优先股
- **其他主要持仓**：通用食品、时代公司等

## 管理哲学

### 核心原则（1983年附录）
1. 虽然形式上是公司，但实际上是合伙人关系
2. 董事会大多数成员持有大量股份
3. 长期经营，不关心季度业绩
4. 资本配置集权，运营管理分权

### 管理架构
- **总部**：仅20余人（1987年）
- **资本配置**：巴菲特负责
- **运营管理**：子公司经理人全权负责

## 巴菲特评价摘录

> "伯克希尔的股东应该知道，我和查理把你们的钱当自己的钱管理。" —— 1983年股东信

> "我们只做自己理解的事，我们的能力圈很窄但很明确。" —— 多次提及

> "资本配置是CEO唯一不可委托的工作。" —— 1987年股东信
