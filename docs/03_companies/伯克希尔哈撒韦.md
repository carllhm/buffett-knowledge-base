## 发展历程

### 纺织时代（1839-1985）
- 1839年：河谷瀑布公司（纺织厂）
- 1955年：伯克希尔与哈撒韦合并
- 1965年：巴菲特取得控制权（股价约$12）
- 1985年：关闭纺织业务

### 转型时代（1967-1980年代）
| 年份 | 事件 |
|------|------|
| 1967 | 收购国家赔偿公司（保险） |
| 1972 | 收购喜诗糖果（通过蓝筹印花） |
| 1983 | 蓝筹印花与伯克希尔合并 |
| 1985 | 关闭纺织，全面转向投资 |
| 1985 | 投资首都城市/ABC（$517.5M） |
| 1987 | 投资所罗门兄弟（$700M优先股） |

## 管理哲学

### 核心原则（1983年附录）
1. 虽然形式上是公司，但实际上是合伙人关系
2. 董事会大多数成员持有大量股份
3. 长期经营，不关心季度业绩
4. 资本配置集权，运营管理分权

### 管理架构
- **总部**：仅20余人（1987年）
- **资本配置**：巴菲特负责
- **运营管理**：子公司经理人全权负责
