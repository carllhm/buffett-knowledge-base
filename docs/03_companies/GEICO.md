# GEICO（政府雇员保险公司）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | GEICO (Government Employees Insurance Company) |
| 行业 | 汽车保险 |
| 成立年份 | 1936年 |
| 总部 | 马里兰州 Chevy Chase |
| 伯克希尔持股 | 约35%-41%（非控股） |
| 投资成本 | 约4,700万美元（1976年前后开始买入） |

## 核心业务

GEICO是美国最大的汽车保险直销公司之一，通过电话和网络直接面向消费者销售保险产品，省去中间代理商环节，实现低成本运营。

## 伯克希尔投资历程

### 买入背景
- **1976年**：GEICO因激进扩张和定价失误濒临破产，股价从高点暴跌
- 巴菲特认定其核心竞争优势（低成本模式）依然完好，开始大举买入
- 时任CEO杰克·伯恩（Jack Byrne）成功扭转局面

### 持仓变化
| 年份 | 持股数 | 备注 |
|------|--------|------|
| 1978 | 720万股 | 主要持仓 |
| 1981 | 720万股 | 持仓稳定 |
| 1987 | 690万股 | 持仓稳定 |

### 投资回报
- **未分配盈利**：1981年约2,000万美元/年
- **分红收入**：稳定现金分红
- **市值增长**：持仓市值从约4,700万美元成本增长至数亿美元

## 核心竞争优势

### 1. 低成本运营模式
- 无代理商佣金，直销模式节省15-20%成本
- 目标客户：政府雇员、军人等低风险群体
- 精细化承保选择

### 2. 效率优势
- **综合成本率**：1983年仅96，远优于行业111
- 保费增长快于行业，同时保持承保盈利
- 投资组合管理出色（Lou Simpson负责）

### 3. 品牌优势
- "15分钟省15%"广告深入人心
- 高续保率客户群

### 管理特点
- 承保纪律严格，不惜削减业务保持盈利
- 成本控制极致，持续压低运营费用率
- 投资保守稳健，避免高风险资产
