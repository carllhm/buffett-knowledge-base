# ISCAR/IMC

## 基本信息

- **首次提及**: 2006年
- **持股情况**: 2006年7月5日收购80%
- **行业**: 金属切削工具制造
- **英文名**: ISCAR / IMC (International Metalworking Companies)
- **总部**: 以色列
- **董事长**: 埃坦·韦特海默（Eitan Wertheimer）
- **CEO**: 雅各布·哈帕兹（Jacob Harpaz）

## 巴菲特论述

2006年7月5日收购80%股权，这是伯克希尔首次海外收购。ISCAR是以色列切削工具公司，在61个国家运营。

后更名为IMC (International Metalworking Companies)，成为伯克希尔"五大非保险业务支柱"之一。

2009年，ISCAR收购日本领先小工具生产商Tungaloy，进一步扩展全球业务。

## 投资意义

这次收购标志着巴菲特开始将投资版图扩展到美国以外。ISCAR代表了高质量的工业制造企业，拥有强大的技术优势和全球市场地位。巴菲特特别赞赏韦特海默家族的管理能力，这是一个典型的"买入并持有"的优秀企业。
