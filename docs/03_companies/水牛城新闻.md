# 水牛城新闻（Buffalo News）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | Buffalo News (原名 Buffalo Evening News) |
| 行业 | 报纸/媒体 |
| 总部 | 纽约州水牛城 |
| 伯克希尔持股 | 100%（通过蓝筹印花收购） |
| 收购年份 | 1977年 |

## 核心业务

水牛城市唯一的日报，周一至周六出版晚报，周日出版周日版。

## 收购故事

### 背景
- 1977年蓝筹印花收购
- 收购后立即推出周日版，挑战原有周日报纸Courier-Express

### 早期困境
- **1977-1982年**：与Courier-Express激烈竞争，持续亏损
- 1981年税前亏损106万美元
- 1982年税前亏损122万美元

### 最终胜利
- **1983年**：Courier-Express关门，成为唯一日报
- 当年实现盈利，利润率约10%

## 核心竞争优势

### 1. 渗透率领先
| 指标 | 数值 | 排名 |
|------|------|------|
| 平日渗透率 | - | 美国前100大报纸第一 |
| 周日渗透率 | 83% | 从十年前63%提升 |

### 2. 内容质量
- **新闻洞占比**：50.9%（行业平均仅35-40%）
- 坚持高质量新闻内容
- 主编Murray Light坚持新闻质量

### 3. 广告效率
- 预印插页收入从360万涨到810万美元（1983年）
- 分类广告强劲

### 关键人物

| 姓名 | 职位 | 贡献 |

- 成本控制严格
