# 本·格雷厄姆（Ben Graham）

## 基本信息

| 属性 | 内容 |
|------|------|
| 全名 | Benjamin Graham |
| 出生 | 1894年 |
| 逝世 | 1976年 |
| 职业 | 投资者、教授、作家 |
| 称号 | "价值投资之父"、"证券分析之父" |

## 核心贡献

### 投资理论
1. **《证券分析》**（1934年，与David Dodd合著）
   - 现代证券分析奠基之作
   - 提出"安全边际"概念

2. **《聪明的投资者》**（1949年）
   - 巴菲特称"最好的投资书"
   - 提出"市场先生"比喻

### 核心概念
| 概念 | 含义 |
|------|------|
| 安全边际 | 买入价格显著低于内在价值 |
| 市场先生 | 市场是服务你的伙伴，不是指导你的导师 |
| 内在价值 | 企业真实价值vs市场价格 |

## 与巴菲特关系

### 师生关系
- 1950年：巴菲特在哥伦比亚大学师从格雷厄姆
- 1954-1956年：巴菲特在格雷厄姆-纽曼公司工作
- 格雷厄姆是巴菲特投资思想的奠基人

### 思想传承
| 格雷厄姆教导 | 巴菲特实践 |
|-------------|-----------|
| 安全边际 | 所有投资决策核心原则 |
| 市场先生 | 独立判断，不被市场情绪影响 |
| 烟蒂投资 | 早期沿用，后期转向质量投资 |

## 巴菲特评价摘录

### 1985年股东信

> "我的投资哲学85%来自格雷厄姆，15%来自费雪。"

### 1987年股东信

> "格雷厄姆教导我：市场先生是你的仆人，不是你的导师。"

### 多次提及

> "《聪明的投资者》是我读过的最好的投资书。第8章和第20章是投资者必读。"

## 核心金句

> "短期看，市场是投票机；长期看，市场是称重机。"

> "投资者最大的敌人很可能就是他自己。"

> "在股市中，要想在预测中赚钱，你必须预测预测者的预测。"

## 影响力

- 培养了一代价值投资者（巴菲特、沃尔特·施洛斯、约翰·博格等）
- 奠定了现代证券分析职业基础
- "安全边际"成为价值投资核心原则

## 在股东信中的年份

1984, 1985, 1987
