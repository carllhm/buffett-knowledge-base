# 资本配置

> "资本配置是CEO最重要的职责。做错了这一点，即使运营再好也是徒劳。"——巴菲特

资本配置能力是巴菲特最核心的管理能力，也是伯克希尔持续增长的关键。

---

## 经典语录

> "我们每收到一美元留存收益，都必须问：留在伯克希尔内部还是分给股东？只有当内部再投资的预期回报高于市场时才应该留存。"

> "资本配置是CEO最重要的职责，也是最容易做错的事情。"

> "我们有一个简单的原则：只投资那些预期回报率超过资本成本的项目。否则就分红或回购股票。"

> "回购股票只有在股价远低于内在价值时才有意义。在股价高估时回购是对股东财富的破坏。"

> "伯克希尔最大的优势之一是：我们可以在全球范围内配置资本，而大多数公司CEO只能在自己熟悉的行业里投资。"
