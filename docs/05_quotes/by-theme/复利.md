# 复利

> "复利是神奇的力量，是世界第八大奇迹。理解它的人赚取它，不理解的人支付它。"——巴菲特

复利是伯克希尔帝国建立的秘密，也是巴菲特最常提及的财富哲学。

---

## 经典语录

> "复利是神奇的力量。23%的年复合增长率持续50年，可以把19美元变成价值数十亿美元。"

> "我们在伯克希尔做的事情很简单：把留存收益以高于原有资本的回报率再投资。复利在发挥作用。"

> "储蓄账户如果以复利方式增长，10%的年利率可以在7年后翻倍。大多数人低估了复利的力量。"

> "喜诗糖果每年产生的现金流，我们重新投入喜诗或其他业务。这种复利效应是伯克希尔增长的秘密。"

> "我之所以富有，不是因为我赚得多，而是因为我让利润长期复利增长。从1965年开始，伯克希尔从未停止过复利的脚步。"
