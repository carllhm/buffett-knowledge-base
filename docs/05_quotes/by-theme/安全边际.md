# 安全边际

> "安全边际是投资的基石。永远不要为了追求最大化收益而牺牲安全边际。"——巴菲特

安全边际来自老师格雷厄姆，是巴菲特60年投资生涯一以贯之的纪律。

---

## 经典语录

> "投资的全部意义在于，在好公司以合理价格买入。但合理价格远低于内在价值时买入，就是安全边际。"

> "我们坚持在有安全边际的情况下投资。当没有安全边际时，我们不会出手，不管市场怎么走。"

> "安全边际不是要你等待完美时机，而是要在价格显著低于价值时果断行动。"

> "如果你的分析是正确的，你就会获得安全边际。安全边际是你犯错的容错空间。"

> "当别人都在冒险的时候，安全边际是你最好的朋友。"——对应2008年金融危机
