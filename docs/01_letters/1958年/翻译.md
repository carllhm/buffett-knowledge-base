# 1958年致合伙人信

沃伦·E·[[沃伦·巴菲特]]
内布拉斯加州奥马哈市安德伍德大道5202号

## 1958年股市概况

一位经营中型投资信托的朋友最近写道："美国人特有的反复无常的性格在1958年发生了重大转变，'亢奋'这个词用来形容股市再合适不过了。"

我认为这总结了过去一年主导股市的心理变化，无论是在业余投资者还是专业投资者层面。过去一年，几乎任何理由都被用来为"投资"股市辩护。毫无疑问，现在股市中反复无常的人比多年来任何时候都多，他们停留的时间将取决于他们认为能多快、多轻松地获利。虽然无法确定他们会继续增加多久从而刺激价格上涨，但我认为可以合理地说，他们停留的时间越长，随后的反应就越大。

我无意预测整体市场——我的努力专注于寻找被低估的证券。然而，我确实相信公众普遍认为投资股票必然获利，这最终会导致麻烦。一旦出现这种情况，我认为即使是低估证券的价格也会受到实质性影响（虽然内在价值不会）。

## 1958年业绩

在去年的信中，我写道：

"相对而言，我们的表现在熊市可能比在牛市更好，因此对上述结果的推论应该考虑到这是一个我们本应表现相对较好的年份。如果大盘出现大幅上涨，只要能跟上指数的涨幅我就很满意了。"

后一句话描述了我们1958年的表现类型，我的预测得到了验证。道琼斯工业平均指数从435上涨到583，加上约20点的股息，整体涨幅为38.5%。全年运营的五个合伙企业取得的业绩平均略好于这个38.5%。基于两年年末的市场价值，它们的收益在36.7%到46.2%之间。考虑到相当一部分资产已经并继续投资于从快速上涨的市场中获益甚少的证券，我认为这些结果相当不错。我将继续预测，在下跌或平稳的市场中我们的表现将高于平均水平，但在上涨的市场中，我们将竭尽全力跟上市场。

## 典型案例

为了让您更好地理解我们的运作方式，我认为有必要回顾1958年的一个具体操作。去年我提到我们最大的持仓占各合伙企业资产的10%至20%。我指出，我们希望这只股票下跌或保持相对稳定，这样我们才能积累更大的持仓，正因如此，这类证券在牛市中可能会拖累我们的相对表现。

这只股票是新泽西州联盟城的Commonwealth Trust Co.。当我们开始购买时，按保守计算，其每股内在价值为125美元。然而，由于种种原因，尽管每股盈利约10美元，该公司完全不支付现金股息，这很大程度上导致其价格约为50美元，处于低迷状态。因此，我们拥有一家管理良好、盈利能力强但以大幅低于内在价值的价格出售的银行。作为新股东，管理层对我们很友好，最终损失的风险似乎很小。

Commonwealth有25.5%的股份由一家更大的银行持有（Commonwealth资产约5000万美元——约为奥马哈第一国立银行的一半），该银行多年来一直希望合并。由于个人原因，这种合并被阻止，但有证据表明这种情况不会无限期持续。因此，我们拥有的组合是：

1. 非常强大的防御特性；
2. 价值以令人满意的速度稳健增长；
3. 有证据表明，尽管可能需要一年或十年，但这些价值最终会得到释放。如果后者成真，价值可能会增长到相当大的数字，比如每股250美元。

在大约一年的时间里，我们成功以平均约51美元的价格获得了该银行约12%的股份。显然，让股价保持平稳对我们非常有利。随着我们持仓的增加，特别是当我们成为第二大股东并拥有足够的投票权值得在任何合并提案中被咨询时，我们的持股价值增加了。

Commonwealth只有约300名股东，平均每月可能只有两笔交易，所以您可以理解为什么我说股市的整体活动对我们某些持仓的价格变动影响很小。

不幸的是，我们在购买时遇到了一些竞争，将价格推高到约65美元，在那个价位我们既不是买家也不是卖家。非常小的买单就会在这种不活跃的股票中产生这种幅度的价格变化，这就解释了为什么我们的投资组合持仓不能有任何"泄露"非常重要。

今年晚些时候，我们成功找到一个特殊情况，可以以诱人的价格成为最大股东，因此我们卖出了持有的Commonwealth股份，获得每股80美元，而当时报价市场约为20%的折扣。

显然，我们仍然可以以50美元的价格耐心地少量购买股票，我会对这样的计划感到非常高兴，尽管去年我们相对于市场的表现看起来会很糟糕。像Commonwealth这样的情形最终实现盈利的年份，在很大程度上是偶然的。因此，任何单一年份的表现作为估计长期结果的基础都有严重的局限性。然而，我相信投资于这类被低估且受良好保护的证券的项目是获得证券长期利润的最可靠方式。

## 当前状况

市场水平越高，被低估的证券就越少，我发现要获得足够数量的有吸引力的投资存在一些困难。我希望增加workout投资的比例，但很难找到合适的条件。

因此，在可能的范围内，我正试图通过在几只被低估的证券中建立大仓位来创造自己的workout。这样的策略应该能够实现我之前的预测——在熊市中表现优于平均水平。我希望基于这一点被评判。如果您有任何问题，请随时提出。

沃伦·E·[[沃伦·巴菲特]]
