# 2007年巴菲特致股东信——思维导图

## 时代背景分析

- **次贷危机爆发**：2007年是次贷危机全面爆发之年，房价下跌导致大量次级贷款违约
- **"退潮才知道谁在裸泳"**：巴菲特在信中犀利点评金融机构的问题
- **美元持续走弱**：2007年美元对欧元从年初约0.76美元跌到年底约1.37美元
- **房地产相关行业受挫**：砖块、地毯、房地产经纪业务均受冲击
