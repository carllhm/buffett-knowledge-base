# 1973年 核心总结

> **一句话总结：** ROE从19.8%回落至17.4%，保险承保利润率开始恶化。更重要的是，巴菲特开始了对华盛顿邮报的大手笔投资，伯克希尔正式从"困境转型公司"蜕变为"长期投资平台"。

## 核心数据

| 指标 | 1973年 | 对比1972年 |
|------|--------|-----------|
| 营业收益 | 1193万美元 | +7.3% |
| ROE | **17.4%** | -2.4个百分点 |
| 每股营业收益 | $12.18 | $11.43 |
| 股价涨幅 | — | 显著下跌 |
| 股票组合未实现亏损 | **超1200万美元** | 处境艰难 |

## 三大业务板块：两好一坏

### 银行业务：持续闪耀

Gene Abegg 继续掌舵，伊利诺伊国民银行再次创下历史新高。即便在定期存款占比超60%、需支付最高利率的情况下，税后营业收益率仍超存款规模2.1%。

银行业务已成为伯克希尔的"压舱石"——无论其他业务如何波动，银行始终在创造超额回报。

### 纺织业务：价格管制的受害者

1973年原材料价格暴涨，但美国生活成本委员会的价格管制让定价无法完全跟随成本。纺织业务的实际利润率被管制人为压低。

巴菲特在年报中直言："价格管制或许削平了行业上行期的盈利峰值，却无法避免行业下行期必然到来的利润低谷。"

这说明他对纺织业务的定位非常清晰：维持但不追加投入，等待时机剥离。

### 保险：危机前的平静

1973年的保险承保利润仍属良好，但多个信号已显示危机将至：

- 大量竞争对手以低费率争抢业务
- 损失准备金存在普遍低估
- 汽车险费率无法覆盖快速上涨的通胀成本

Jack Ringwalt 在这一年卸任，Phil Liesche 接任。交接顺利，理念延续。

## 两项重大战略决策

### ① 收购多元零售公司

伯克希尔与多元零售公司（Mutual Berkshire）合并，发行19.5万股伯克希尔股票（净增约8.5万股）。多元零售旗下最重要的资产是：**持有蓝筹印花公司16%的股权**。

这笔收购的战略意义：伯克希尔通过多元零售，间接持有蓝筹印花的少数股权，从而与蓝筹印花建立了更深的资本纽带。多年后，蓝筹印花成为伯克希尔体系内的核心控股平台。

### ② 大手笔增持华盛顿邮报

年报透露，巴菲特在1973年大规模增持了华盛顿邮报的股票。截至年末，持仓成本超1060万美元，持股467,150股B类股。

巴菲特写道："我们计划**永久持有**该笔投资。"

这是伯克希尔历史上第一次明确宣布"永久持有"策略。

## 关键洞察

**洞察一：未实现亏损≠投资失败**

1973年末，股票组合出现超1200万美元的未实现亏损。但巴菲特的判断是：按成本价计量，这些股票具备良好的投资价值，长期将带来稳健收益。

他明确区分了**市场波动**（不重要）与**永久损失风险**（重要）。这段话是后来巴菲特阐述"风险"定义的核心思想雏形。

**洞察二：ROE下降是资本扩张的必然代价**

巴菲特坦诚解释：股东权益规模增长速度快于收益增长，ROE必然下降。这是他有意为之的资本配置决策，不是管理失误。

提前向股东预警业绩可能下滑，并说明原因——这种坦诚沟通，成为伯克希尔年报文化的重要特征。

**洞察三：蓝筹印花是关键连接点**

1973年的这笔合并，让蓝筹印花成为伯克希尔体系的核心节点：
- 伯克希尔 → 多元零售 → 蓝筹印花（16%）
- 蓝筹印花 → 喜诗糖果（1972年收购）

这条资本纽带，在未来20年持续强化，最终让蓝筹印花在1983年完全并入伯克希尔体系。

## 名言

> "我们预计，该组合长期将带来稳健的收益。"
> ——沃伦·巴菲特，1973年年报，论股票投资组合

（解读："长期"是关键词。面对1200万美元的账面亏损，巴菲特的回应是等待——相信企业内在价值会最终反映在市场价格上。这与后来那句名言一脉相承："如果你不打算持有一只股票十年，那就不要持有十分钟。"）

## 历史坐标

- 距离1956年巴菲特合伙基金：**17年**
- 距离1965年接手伯克希尔：**8年**
- **开始大手笔投资华盛顿邮报**，这笔投资最终持有40年，收益率超过百倍
- 距离1974年灾难性保险亏损：**1年**
