# 蓝筹印花（Blue Chip Stamps）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | Blue Chip Stamps |
| 行业 | 印花业务/投资控股 |
| 总部 | 加利福尼亚州洛杉矶 |
| 伯克希尔持股 | 约58%（1983年合并前） |
| 合并年份 | 1983年 |

## 核心业务

### 起源
- 1956年成立，提供零售印花业务
- 消费者购物获赠印花，收集后兑换商品
- 1960年代高峰期年收入超1.2亿美元

### 业务衰退
- 1970年代后印花业务持续萎缩
- 但公司持有大量浮存金可用于投资

## 投资控股转型

### 关键收购
| 年份 | 收购标的 | 价格 | 备注 |
|------|---------|------|------|
| 1972 | [喜诗糖果](./喜诗糖果.md) | 2,500万美元 | 经典案例 |
| 1971 | 水牛城晚报 | - | 后全资持有 |
| - | 韦斯考金融80%股权 | - | 间接持有 |

### 投资组合
- 持有喜诗糖果100%
- 持有水牛城新闻100%
- 持有韦斯考金融80%
- 持有其他小企业股权

## 与伯克希尔合并

### 合并背景
- 1983年年中完成合并
- 换股比例：蓝筹印花股东获得伯克希尔股份
- 查理·芒格任董事长（合并前）

### 合并原因
1. 消除交叉持股复杂性
2. 简化财务报告
3. 整合投资决策
