# 美国运通（American Express）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | American Express Company |
| 行业 | 金融（支付/信用卡） |
| 成立年份 | 1850年 |
| 总部 | 纽约 |
| 伯克希尔持股 | 约17-20%（最大股东） |
| 投资成本 | 约14亿美元（1964年起陆续买入） |

## 核心业务

美国运通是全球领先的支付和旅行服务公司，主要业务包括：
- 信用卡发行
- 旅行服务
- 预付费卡
- 企业支付解决方案

## 伯克希尔投资历程

### 买入背景
- **1964年**：巴菲特开始买入美国运通
- 当时公司因"色拉油丑闻"股价暴跌
- 巴菲特认定其信用卡业务有巨大价值

### 投资规模
- 初始投资：约1400万美元
- 1994年：持仓约3370万股，成本约14亿美元
- 1994年市值：约28亿美元

### 投资回报
- 持有超过60年
- 是伯克希尔最长情的投资之一
- 累计分红数十亿美元

## 核心竞争优势

### 1. 品牌特权
- "不能刷美国运通"是商户的耻辱
- 高端品牌形象深入人心
- 支付行业最强品牌之一

### 2. 支付网络
- 全球最大的独立支付网络
- 商户覆盖广泛
- 网络效应强大

### 3. 客户基础
- 高净值客户群体
- 商户关系稳固
- 续费率高

## 巴菲特评价摘录
