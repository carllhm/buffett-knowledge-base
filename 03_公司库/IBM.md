# IBM（国际商业机器）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | International Business Machines (IBM) |
| 行业 | 科技/IT服务 |
| 成立年份 | 1911年 |
| 总部 | 纽约阿蒙克 |
| 伯克希尔持股 | 约5.5%（2011年买入，2018年清仓） |
| 投资成本 | 约130亿美元 |

## 核心业务

IBM是全球领先的IT服务和咨询公司，主要业务包括：
- IT咨询服务
- 云服务
- 认知计算（Watson）
- 企业软件
- 硬件（服务器、存储）

## 伯克希尔投资历程

### 买入背景
- **2011年**：巴菲特开始买入IBM
- 当时IBM正在进行转型
- 看中其服务业务的护城河

### 持仓变化
- 2011年：开始买入
- 2014年：持仓达到峰值约6400万股
- 2017-2018年：全部清仓

### 投资回报
- 成本：约130亿美元
- 清仓时：约130亿美元
- 基本持平，无收益
