# 内布拉斯加家具城（Nebraska Furniture Mart）

## 基本信息

| 属性 | 内容 |
|------|------|
| 英文名 | Nebraska Furniture Mart (NFM) |
| 行业 | 家具/家居零售 |
| 成立年份 | 1937年 |
| 总部 | 内布拉斯加州奥马哈 |
| 伯克希尔持股 | 90%（1983年收购） |
| 收购价格 | 约5,500万美元（多数股权） |

## 核心业务

美国最大的单店家居零售商，主营家具、地毯、电器等。单店年销售额超1亿美元，超过奥马哈所有竞争对手总和。

## 收购故事

### 背景
- B夫人（Rose Blumkin）1937年用500美元创办
- 1983年单店年销售额已超1亿美元
- 巴菲特长期观察，了解其竞争力

### 收购条款
- 1983年收购90%股权
- B夫人家族保留10%
- 关键年轻经理获得10%期权激励
- **无审计、无库存盘点、无资产核实**

### 收购原因
1. **极致低成本运营**：毛利率仅20%+，运营费用率16.5%
2. **采购优势**：Louie Blumkin是全美最精明采购员
3. **管理传奇**：B夫人90岁仍一周七天在卖场
4. **本地垄断**：远超所有竞争对手效率

## 核心竞争优势

### 1. 成本优势
- 运营费用率仅16.5%（行业标杆Levitz为35.6%）
- 极低毛利率但有盈利能力
- 采购规模议价权强

### 2. 管理效率
- 无复杂管理系统
- 家族式经营，决策快速
- 员工忠诚度高

### 3. 品牌效应
- 奥马哈本地绝对领先
- 口碑传播，获客成本低

### B夫人传奇
- **1893年**：生于俄国，未受过正规教育
- **1937年**：500美元创办家具店
- **1983年**：90岁，一周七天在卖场卖地毯
- **1984年**：91岁仍每周工作七天
- **1987年**：94岁仍在工作
- **格言**："低价卖，说实话"
