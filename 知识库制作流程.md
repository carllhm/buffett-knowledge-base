# 巴菲特知识库制作流程

## 项目概览

本项目是一个完整的巴菲特70年致股东信知识库，包含：
- 原文翻译（1956-2024年）
- 核心总结、思维导图
- 主题索引、公司库、人物库
- 金句库、数据可视化、扩展阅读

## 五、构建与部署

2. 选择 "Connect to Git" → 选择仓库

## 七、常见问题与解决方案

### 问题 1：Wiki-link 指向搜索而不是概念页
**原因：** 缺少 CONCEPT_MAP 映射
**解决：** 在 config.mts 中添加：
```typescript
const CONCEPT_MAP: Record<string, string> = {
  '内在价值': 'intrinsic-value',
  // ...
}
```

### 问题 2：Cloudflare Pages 嵌套路由 404
**原因：** VitePress 默认路由配置问题
**解决：** 添加 buildEnd 钩子生成 `_redirects` 文件

### 问题 3：年份侧边栏配置太长
**解决：** 按时期分组（70年分6个时期），折叠显示
